### Project Title: Titanic Survival Prediction
This project focuses on the classic Titanic dataset to predict passenger survival based on various features such as age, gender, and ticket class.

###  Dataset Description
The dataset contains information about the passengers aboard the Titanic. Key features include:
* **Pclass:** Passenger class (1st, 2nd, or 3rd).
* **Sex:** Gender of the passenger.
* **Age:** Age in years.
* **Survived:** Target variable (0 = No, 1 = Yes).

###  Machine Learning Task
* **Task Type:** Classification
* **Objective:** To build a model that classifies passengers into survivors or non-survivors based on their demographic and travel information.
